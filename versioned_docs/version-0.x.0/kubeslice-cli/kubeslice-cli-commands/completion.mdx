# Completion [work-in-progress]
Generate the autocompletion script for kubeslice-cli for the specified shell.

## Usage
kubeslice-cli completion [command][flags]

## Commands
The following are the commands:

|Command|Description|
|----|----|
|bash|Generate the autocompletion script for bash.|
|fish|Generate the autocompletion script for fish.|
|powershell|Generate the autocompletion script for powershell.|
|zsh|Generate the autocompletion script for zsh.|

## Flags
The following are the `kubeslice-cli completion` flags:

|Name|Shorthand|Usage|
|----|----|----|
|help|-h|The help for completion.|
|config|-c| **Global flag**. The path to the topology configuration YAML file. |
