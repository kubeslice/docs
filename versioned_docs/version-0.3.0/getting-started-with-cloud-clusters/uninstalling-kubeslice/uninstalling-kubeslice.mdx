# Uninstalling KubeSlice

Introduction
This topic describes the steps to uninstall the KubeSlice. To uninstall the KubeSlice 
Controller, follow these steps:

* Offboard all namespaces from the slice
* Delete the slice
* Deregister your worker cluster
* Delete the project
* Uninstall the KubeSlice Controller

:::info
The order of operations in uninstalling the KubeSlice Controller is important to ensure 
all resources are cleaned up properly. Follow the order of operations provided under 
this topic.
:::
